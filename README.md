 🏨 Hospitality Analytics Dashboard

---

## 📌 Project Overview

This project focuses on analyzing hospitality industry data using **Power BI** to uncover key business insights related to revenue, occupancy, and performance across different properties.

The dashboard enables stakeholders to monitor hotel performance, identify trends, and make **data-driven decisions** to improve profitability and operational efficiency.

---

## 🎯 Objectives

- Analyze hotel performance across multiple properties  
- Track key metrics like **Revenue, Occupancy, and ADR (Average Daily Rate)**  
- Identify trends and patterns in bookings and customer behavior  
- Enable interactive and dynamic data exploration  
- Support strategic decision-making in the hospitality domain  

---

## 🛠️ Tech Stack

- **Power BI** – Data Visualization & Dashboard  
- **DAX** – Measures & Calculated Columns  
- **Excel / CSV** – Data Source  

---

## 📊 Key Features

- Interactive dashboard with dynamic filters  
- KPI tracking (Revenue, Occupancy %, ADR, RevPAR)  
- Property-wise and city-wise performance analysis  
- Time-based trend analysis  
- Clean and intuitive data visualization
